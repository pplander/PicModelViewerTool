# 万能模型浏览工具 (PicModelViewerTool)

## 技术方案

### 库版本
- **Qt**: 6.10.1 MSVC2022 64-bit (`D:\qt\6.10.1\msvc2022_64`)
- **OSG**: 3.6.5 (`E:\Program\osg\osg`) - 含osgQOpenGL模块
- **Assimp**: 6.0.4 (`E:\Program\assimp\assimp-6.0.4`) - vc143编译

### 架构设计
- 自定义QOpenGLWidget子类嵌入OSG渲染（兼容Qt6）
- Assimp加载OSG不支持的格式，转换为osg::Node
- Qt翻译系统(QTranslator)实现中英文切换
- CMake构建系统

## 实现任务

### Task 1: CMake构建系统
- 创建 `CMakeLists.txt`，配置Qt6/OSG/Assimp路径
- 设置MSVC2022 64-bit编译器
- 配置头文件搜索路径和库链接
- 配置OSG插件路径环境变量（运行时）
- 配置翻译文件生成规则(lupdate/lrelease)

### Task 2: OSG渲染控件 (OSGWidget)
- 继承QOpenGLWidget，实现OSG嵌入渲染
- 创建 `osgViewer::GraphicsWindowEmbedded` 作为OSG图形窗口
- 实现 initializeGL / resizeGL / paintGL
- 集成 osgGA::TrackballManipulator 实现轨道相机控制
- 支持鼠标旋转/平移/缩放
- 添加线框/实体/点云显示模式切换

### Task 3: 模型加载器 (ModelLoader)
- 优先使用OSG osgDB加载支持的格式(.osg/.osgb/.obj/.3ds/.dae/.stl等)
- 对OSG不支持的格式(.fbx/.gltf/.glb/.ply等)使用Assimp加载
- 自动检测文件格式选择加载器
- 加载后自动计算包围盒并调整相机

### Task 4: Assimp到OSG转换器 (AssimpToOSGConverter)
- 将aiScene转换为osg::Node树
- 转换网格数据(aiMesh → osg::Geometry)
- 转换材质(aiMaterial → osg::StateSet)
- 转换纹理坐标和纹理贴图
- 递归转换节点层次(aiNode → osg::MatrixTransform)
- 转换法线和颜色数据

### Task 5: 主窗口 (MainWindow)
- QMainWindow布局：中心OSGWidget + 多个DockWidget
- 菜单栏：文件/视图/工具/语言/帮助
- 工具栏：打开/重置视图/显示模式/截图
- 状态栏：显示顶点数/面数/FPS
- 拖放文件打开支持
- 最近打开文件记录

### Task 6: 模型信息面板 (ModelInfoDock)
- 显示文件名/路径/大小/格式
- 显示顶点数/面数/网格数/材质数
- 显示包围盒尺寸
- 显示动画信息（如有）

### Task 7: 场景树面板 (SceneTreeDock)
- QTreeView展示OSG场景节点层次
- 点击节点高亮/聚焦
- 显示节点类型/名称

### Task 8: 光照控制面板 (LightControlDock)
- 开关光源
- 调整环境光/漫反射/镜面反射颜色
- 调整光源位置
- 双面光照开关

### Task 9: 国际化 (I18nManager)
- QTranslator管理中英文翻译
- 默认中文，菜单可切换
- 运行时动态切换语言
- 所有UI文本使用tr()包裹

### Task 10: 资源文件与图标
- 创建resources.qrc
- 添加应用图标
- 配置Windows应用程序图标(.rc)

## 支持的模型格式

| 来源 | 格式 |
|------|------|
| OSG原生 | .osg, .osgb, .osgt, .ive |
| OSG插件 | .obj, .3ds, .dae, .stl, .ply, .jpg/png/bmp(天空盒) |
| Assimp扩展 | .fbx, .gltf, .glb, .amf, .3mf, .off, .ac, .bvh, .csm, .dxf, .hmp, .lwo, .lws, .md2, .md3, .md5, .mdc, .mdl, .ms3d, .nff, .smd, .ter, .x, .xgl, .zgl |
